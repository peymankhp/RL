📐 RL Formulas Reference — DP · MC · TD

1. Transition Dynamics

Transition Probability
$$P(s', r \mid s, a) = \Pr\{S_t = s',\, R_t = r \mid S_{t-1} = s,\, A_{t-1} = a\}$$
Markov Property
$$P(S_{t+1} \mid S_t, S_{t-1}, \ldots, S_0) = P(S_{t+1} \mid S_t)$$

2. Expected Return

Return

$$G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$$

State-Value Function

$$v_\pi(s) = \mathbb{E}_\pi\!\left[G_t \mid S_t = s\right] = \mathbb{E}_\pi\!\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \;\middle|\; S_t = s\right]$$

Action-Value Function

$$q_\pi(s, a) = \mathbb{E}_\pi\!\left[G_t \mid S_t = s,\, A_t = a\right] = \mathbb{E}_\pi\!\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \;\middle|\; S_t = s,\, A_t = a\right]$$

3. Dynamic Programming

Bellman Expectation Equation

$$V^\pi(s) = \sum_a \pi(a \mid s) \sum_{s', r} P(s', r \mid s, a)\,\bigl[r + \gamma V^\pi(s')\bigr]$$
⚠️ $\pi(a \mid s)$ is the probability of taking action $a$ in state $s$ under policy $\pi$.

Bellman Optimality Equation

$$V^*(s) = \max_a \sum_{s', r} p(s', r \mid s, a)\,\bigl[r + \gamma V^*(s')\bigr]$$

Policy Evaluation Update

$$V_{k+1}(s) = \sum_a \pi(a \mid s) \sum_{s', r} p(s', r \mid s, a)\,\bigl[r + \gamma V_k(s')\bigr]$$

4. Monte Carlo Methods

💡 MC methods require no model of the environment — only sample episodes. Uses $P(s' \mid s, a)$ implicitly through interaction.

Return (Monte Carlo)

$$G_t = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$$

State Value (MC Estimate)

$$v_\pi(s) \approx \text{average of returns observed after visiting } s$$

Action Value (MC Estimate)

$$q_\pi(s, a) \approx \text{average return after taking action } a \text{ in state } s$$

5. Temporal-Difference (TD) Prediction

💡 TD combines the sampling of MC with the bootstrapping of DP. Updates after every step, not every episode.

TD(0) Update Rule

$$V(S_t) \leftarrow V(S_t) + \alpha\bigl[R_{t+1} + \gamma V(S_{t+1}) - V(S_t)\bigr]$$

TD Error

$$\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$$

The TD error $\delta_t$ measures how much the current value estimate needs to be corrected.

6. TD Error Derivation — Connecting MC Return to TD Errors

Starting from the MC return and the recursive definition:

Recursive return
$$G_t = R_{t+1} + \gamma G_{t+1}$$
Step 1 — Expand the error
$G_t - V(S_t)$
$= R_{t+1} + \gamma G_{t+1} - V(S_t)$
$= R_{t+1} + \gamma V(S_{t+1}) - V(S_t) + \gamma\bigl(G_{t+1} - V(S_{t+1})\bigr)$
$= \delta_t + \gamma\bigl(G_{t+1} - V(S_{t+1})\bigr)$    where $\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$
Step 2 — Unrolling
$= \delta_t + \gamma\delta_{t+1} + \gamma^2\bigl(G_{t+2} - V(S_{t+2})\bigr)$
$= \delta_t + \gamma\delta_{t+1} + \gamma^2\delta_{t+2} + \cdots$
Final Result
$$G_t - V(S_t) = \sum_{k=t}^{T-1} \gamma^{k-t}\,\delta_k$$
✅ This identity shows that the MC error (return minus value) equals the discounted sum of all future TD errors. It is the foundation of eligibility traces and TD(λ).
